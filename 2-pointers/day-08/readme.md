# Day 08 - Two Pointers

## Overview

Today I learned the **Two Pointers Technique**, a popular algorithmic pattern used to solve array and string problems efficiently.

The main idea is to use two indices (pointers) that move through the data structure based on specific conditions, often reducing the time complexity from **O(n²)** to **O(n)**.

## Concepts Covered

* Two Pointers Technique
* Opposite Direction Pointers
* Same Direction Pointers
* Time Complexity Optimization
* Pair Finding Problems
* Palindrome Checking

## Key Takeaways

* Two Pointers is a technique, not a data structure.
* Commonly used with arrays and strings.
* Helps eliminate unnecessary nested loops.
* Frequently appears in coding interviews and competitive programming.
