### 1. LeetCode 1047 - Remove All Adjacent Duplicates In String
- **Difficulty:** Easy
- **Status:** ✅ Solved
- **Concepts Used:**
  - Stack
  - String Traversal
  - Push & Pop Operations
  - LIFO (Last In, First Out)

**Approach:**
- Traverse the string character by character.
- If the current character matches the top of the stack, remove the top element.
- Otherwise, push the current character onto the stack.
- Join the remaining characters in the stack to form the final string.

**Time Complexity:** `O(n)`

**Space Complexity:** `O(n)`
