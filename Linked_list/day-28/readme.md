## ✅ Problem Solved

* **LeetCode 203** - Remove Linked List Elements *(Easy)*

## 🎯 What I Learned

* How to remove nodes from a singly linked list.
* Why deleting the head node requires special handling.
* The purpose of using a **dummy node** to simplify edge cases.
* How to manipulate pointers without breaking the linked list.
* Why `previous` should only move when the current node is **not** deleted.

## 💡 Key Takeaways

* Linked Lists are all about updating pointers correctly.
* A dummy node helps avoid separate logic for deleting the head.
* Deleting a node means **changing links**, not deleting memory.
* Careful pointer movement prevents skipping nodes or creating bugs.
