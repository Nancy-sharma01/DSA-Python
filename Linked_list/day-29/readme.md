## 💡 Key Learnings
- Reversing a linked list requires changing the direction of every `next` pointer.
- Always save `curr.next` before modifying it.
- Move pointers in the correct order to avoid losing the remaining list.
- The reversed list starts at `prev` after the traversal is complete.

## ⏱️ Time Complexity

- **Space:** O(1)
