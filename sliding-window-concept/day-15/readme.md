# Day 14 – Sliding Window 🚀

## 📌 Problems Solved

### 1. LeetCode 209 – Minimum Size Subarray Sum

**Difficulty:** Medium

**Concepts Learned:**

* Variable Sliding Window
* Expanding and shrinking the window dynamically
* Maintaining the current window sum
* Updating the minimum window length before shrinking
* Handling edge cases when no valid subarray exists

**Time Complexity:** `O(n)`
**Space Complexity:** `O(1)`

---

### 2. LeetCode 3 – Longest Substring Without Repeating Characters

**Difficulty:** Medium

**Concepts Learned:**

* Variable Sliding Window
* Using a `set` to track characters in the current window
* Removing duplicate characters by shrinking the window
* Maintaining the longest valid substring without repeating characters

**Time Complexity:** `O(n)`
**Space Complexity:** `O(min(n, m))`, where `m` is the size of the character set.

---
