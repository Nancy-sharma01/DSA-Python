## ✅ Problem Solved

### LeetCode 217 - Contains Duplicate (Easy)

**Concepts:**
- Hash Set
- Duplicate Detection

**Approach:**
- Traverse the array while storing elements in a Hash Set.
- If an element is already present in the set, return `True`.
- Otherwise, add it to the set.
- If no duplicates are found, return `False`.

**Time Complexity:** `O(n)`

**Space Complexity:** `O(n)`

---

## 💡 Key Learning

Hash Sets provide average `O(1)` lookup time, making them ideal for duplicate detection problems.
