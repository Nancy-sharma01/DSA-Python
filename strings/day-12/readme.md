## 🧠 What I Learned Today

- Strings in Python are immutable, so converting them to a list is useful for in-place modifications.
- Breaking a problem into smaller blocks often leads to simpler solutions.
- Python's built-in string methods (`split()` and `join()`) make string manipulation clean and efficient.
- Reading the problem statement carefully is just as important as writing the code.
