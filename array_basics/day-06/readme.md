## Problems Solved

### 1. Valid Palindrome (LeetCode 125)
**Difficulty:** Easy

**Concepts Used**
- Two Pointers
- String Processing
- Alphanumeric Character Checks

**Approach**
- Start pointers from both ends.
- Skip non-alphanumeric characters.
- Compare characters after converting them to lowercase.
- Return False on mismatch.

---

### 2. Best Time to Buy and Sell Stock (LeetCode 121)
**Difficulty:** Easy

**Concepts Used**
- Arrays
- Greedy Approach
- Single Pass Traversal

**Approach**
- Track the minimum stock price seen so far.
- Calculate profit at each step.
- Maintain the maximum profit encountered.
